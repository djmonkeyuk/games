Tools are convenient objects you can add to the scene that can allow you to add parts in a faster and procedural way.

You can find these tools in the "Tools" section of the Part Drawer at the bottom of the screen.

![alt text](/images/nms-bba/guides/guide_tools_1.png)

Tools are shown as red in the scene.

Selecting a tool in the scene will present their own options in the properties panel that allow you to configure their settings.

The settings are slightly different depending on which tool type is selected.

![alt text](/images/nms-bba/guides/guide_tools_2.png)

The `Paste Objects to Scene` button allows you to paste a copy of each base part represented by the tool into the scene, allowing you to further tweak individual base parts away from the tool. Tools are supported by the `Export to NMS` function, so pasting objects in this was is completely optional.

![alt text](/images/nms-bba/guides/guide_tools_6.png)
