## 🛠️ Version 1.0.1

### Updates
* The settings menu has been visually reorganised and split into categories to accommodate new options (for this update, and more that may be introduced in the future).
* Players can now adjust screen resolution and fullscreen/window options within the settings menu.
* Note: The game has a responsive UI with an emphasis on text legibility, personally I have found this working best on Windowed or Windowed Fullscreen modes, but the option to change it is now there for your preference.

* Changed Noneye Desert footprint sound to be more appropriate for sand.

### Bugfixes
* Fixed an issue where the house variants in the city building options were not using up the resources in Lucas' storage box. (sorry, no more free houses!)
* Fixed an issue where character creation scroll bars were not responding to the right thumbstick when using a controller.
* Fixed an issue where footsteps could not be heard around the Worm's Tail beach and water areas.
* Fixed a rare instance where font legibility would be sub-optimal in high DPI cases.
* Fixed a rare instance where it would not be possible to return from procedurally generated areas due to the return sign post being obstructed by an object.
* Fixed an issue where the watering can region highlights would not disappear after cancelling a pending watering action.

### Demo
* All relevant updates have also been applied to the demo too. 🐝
